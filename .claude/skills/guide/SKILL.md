# Framework Context

When guiding in the framework context, the primary sources are `core/nla-foundations.md`
(especially the Working Rhythms section) and `CLAUDE.md` (the skills table). There is no
`app/overview.md` in the framework itself — the framework is infrastructure, not a domain
project. Focus on explaining the framework's purpose, the skill catalog, and how domain
projects relate to the framework.

Read and follow `core/skills/guide.md`.
