# My NLA Project

A Natural Language Application built on the [NLA Framework](../nla-framework/).

## Prerequisites

The NLA Framework must be available at `../nla-framework/` (sibling directory to this project).

## Quick Start

1. Start Claude Code in this directory
2. Run `/startup` to load foundational context
3. Run `/format-article` and provide content
4. Review the output
